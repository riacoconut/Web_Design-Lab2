**Дисципліна:** Web-дизайн

**Тема:** Розробка функціональності Web-додатка мовою Javascript

## Завдання

Розробити функціональність для статичних сторінок Web-додатку першої лабораторної роботи із використанням шаблону MVC.

## Опис додатку

**AlarmX** — це веб-додаток для керування будильниками. Дозволяє користувачам реєструватися, входити в обліковий запис, додавати, вмикати/вимикати та видаляти будильники. Дані зберігаються у `localStorage`.

## Сторінки

| Файл | Опис |
| --- | --- |
| `index.html` | Робоча сторінка зі списком будильників (реалізована за шаблоном MVC) |
| `about.html` | Сторінка з описом додатку |
| `login.html` | Вхід та реєстрація користувача |
| `profile.html` | Профіль користувача |

## Структура MVC робочої сторінки

- `js/alarms-model.js` — **Model**: дані будильників і робота з `localStorage`.
- `js/alarms-view.js` — **View**: відмалювання списку та годинника, обробка форми.
- `js/alarms-controller.js` — **Controller**: зв'язує Model і View, перевіряє спрацьовування будильників.
- `js/app.js` — точка входу, що створює Model, View та Controller.

## Допоміжні модулі

- `js/storage.js` — обгортка над `localStorage` для користувачів та сесії.
- `js/nav.js` — показ/приховання пунктів меню залежно від авторизації.
- `js/auth.js` — реєстрація та вхід.
- `js/profile.js` — відображення даних користувача.
